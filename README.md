# Multi-Agent AI Workflow System

Production-grade multi-agent system built using LangGraph and LangChain,
featuring tool-calling, structured outputs, and supervisor-style orchestration.

## Features
- ReAct-based tool calling
- Hugging Face cloud LLMs
- Explicit state management
- Multi-agent orchestration
- Structured report generation
- Testable & extensible

## Setup
```bash
pip install -r requirements.txt
export HUGGINGFACEHUB_API_TOKEN=your_token
python app/main.py
